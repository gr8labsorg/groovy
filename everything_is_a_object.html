Everything is a Object

Goal

All types are objects in Groovy.

Because Groovy is so closely related to Java, we can use all Java types in Groovy. So we don't experience any difference if we use a primitive type in our Groovy code compared to Java. But Groovy takes it a step further, because in Groovy everything is an object. Primitive types are automatically boxed to their object counterparts. And Groovy will automatically unbox the type if necessary. 

// Define a variable of primitive type int.
int number = 100

// Class is java.lang.Integer.
assert number.class.name == 'java.lang.Integer'

// Other primitive types.
double doubleValue = 1.2080
boolean booleanValue = true
char charValue = 'G'

assert doubleValue.class.name == "java.lang.Double"
assert booleanValue.class.name == "java.lang.Boolean"
assert charValue.class.name == "java.lang.Character"
